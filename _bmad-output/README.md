# BMAD output directory

Configured in `_bmad/bmm/config.yaml` as `output_folder` for this project.

## Layout

- **project-context.md** — Agent implementation rules (generate/update via `bmad-bmm-generate-project-context`).
- **planning-artifacts/** — PRD, architecture, UX, and related planning outputs from BMM workflows.
- **implementation-artifacts/** — Sprint status, story files, and implementation-phase outputs.
- **test-artifacts/** — Reserved for TEA / test-architecture workflows.

## Project knowledge

Human-oriented documentation lives under **`docs/`** (see `docs/index.md`). Dense LLM-oriented bundles live under **`docs/docs-distillate/`**.
