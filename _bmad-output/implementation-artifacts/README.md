# Implementation artifacts (BMM)

Configured in **`_bmad/bmm/config.yaml`** as **`implementation_artifacts`**.

Populate with **`bmad-bmm-sprint-planning`** (`sprint-status.yaml`, etc.) and per-story files from **`bmad-bmm-create-story`** / **`bmad-bmm-dev-story`**.

Until then, execution backlog references **`docs/stories/STORY-*.md`**.
